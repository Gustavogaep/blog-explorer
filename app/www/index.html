Topic Model

A Structural Topic Model was fitted on the corpus of 30+K blog articles. The charts summarise the resulting 60 topics and their correlation between each other. Structural Topic Models also allow to estimate the proportion of each document about a topic based on document-specific covariates, in this case the publication date of an article.

Project

Standing on the shoulders of giants...
Inspiration

Julia Silge's fantastic blog and her article on STM are the foundation of this project.

The blog article of Andrea De Angelis gave inspiration to exploit meta information in topic models.

The idea of a topic correlation graph came from the vignette of the stminsights package.

Visualisations

The beautiful ECHARTS engine is behind all charts used in this project. The engine is very comprehensive, supports many chart types and comes with a selection of stylish themes. John Coene developed an easy to use R wrapper complete with Shiny support, documentation and impressive examples.

The network visualations are powered by DataStorm's visNetwork package.

Key Technology

The topic models are generated using the stm package. Papers on the topic can be found here. Some code from the tidystm package was used to transform STM outputs into tidy format.

The data was scraped using the super convenient rvest package.

The development environment of this project is a Rocker based Docker container running the Open Source version of RStudio Server.

Themes & Assets

The amazing Titan HTML theme is developed by ThemeWagon and can be downloaded for free!

The hero image on the top of this application was created by Callum Wale and made available at Unsplash for free.

I used HTML templates to create a unique look for this Shiny application. The code is structured using Shiny modules.
